# AI-Ticket-Automation
##Project Overview

Customer support teams receive a large number of requests every day through emails, forms, or support portals. These requests must be manually reviewed, categorized, and prioritized before being assigned to the appropriate team.
Manual processing slows down response times and increases operational workload.
This project demonstrates how internal support processes can be automated using workflow automation and data analytics tools. The system automatically processes support tickets, categorizes them, assigns priorities, and provides analytics dashboards for monitoring trends.
The automation workflow is implemented using n8n and the analytics dashboard is built using Microsoft Power BI.
